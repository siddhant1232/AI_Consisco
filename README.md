## Documentation

### Overview
Conciso is an AI-powered PDF summarization web application that allows users to upload PDF documents and receive concise summaries generated using AI models.

### How It Works
1. User uploads a PDF file through the web interface
2. The file is processed on the client and sent to the backend API
3. The AI service analyzes the document content
4. A summarized version is returned and displayed to the user

### Inputs
- PDF file (up to 50+ pages)

### Outputs
- Text-based summarized content
